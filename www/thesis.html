4.1
Introduction
Image segmentation is a field within computer vision which seeks to auto-
matically separate objects within an image similarly to how the human visual
system does(Ballard and Brown, 1982). There are two main types of images,
colour and grayscale. For grayscale images, computers identify each pixel
within an image by a single continuous number over a [0,
1] range indicat-
ing brightness. Similarly, colour image pixels are represented by a measure of
brightness for each colour channel (RGB colour space: red, green, and blue or
CMYK colour space: cyan, magenta, yellow and black) by 8-bit integers over
a [0,
255] range. Pixel colour intensities can also be represented as continuous
value over [0, 1] range as well. Pixels represented by zeros appear in a picture
as black and values of ones (255 in 8-bit) as white. Clustering algorithms utilise
relational structures based on pixel intensities to separate objects within an
image. Preprocessing tools can assist with defining these relationships within
an image.
Before an image can be analysed, each pixel location must be indexed so
as to retain image structure liken to the human eye. When images are of
high quality, containing many pixels, sampling is suggested in order to reduce
computational expense. Sampling pixels can avoid overfitting a solution and
produce better final results. Once image data is indexed and possibly sampled,
further processing may enhance the image segmentation.
Minimum density hyperplane (MDH) segments an image by associating
distinct clusters based on observed densities. The solution plane minimises
the integrated density. This amounts to segmenting dense regions of pixel in-
tensities within an image. Consider a hypothetical scenario in which a picture
consists of a circle filled in with red
